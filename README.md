## 🏭 What I'm building

**[Panama in Context](https://github.com/wanderindev/ai-powered-content-generation-engine) — an AI content factory.**
A bilingual educational platform where one topic suggestion fans out through parallel AI pipelines into published articles, narrated lesson videos, study guides, and social posts. Agents do the work; a human approves at every gate. Claude Sonnet + Haiku, claims-level fact-checking, DeepL translation, ElevenLabs narration, and a from-scratch narrated-video pipeline (Chromium → ffmpeg). FastAPI · React · PostgreSQL.

**[PIC Companion](https://github.com/wanderindev/research-companion-extension) — an API around a tool that doesn't have one.**
A Chrome extension that drives Claude's deep-research mode unattended, turning a browser UI into an automation surface that feeds the content factory.

## 🤖 How I work with AI

I don't treat AI as a helper bolted onto a human-paced process. I build the **leverage** that lets agents own whole workflows &mdash; the skills, context, guardrails, and self-verifying checks &mdash; and keep humans in the loop where judgment matters. Nothing ships open-loop.

I'm currently leading my team's AI-adoption effort: moving us from "AI writes code faster" to AI woven across the core workflows we run every day &mdash; safely, with shared tooling and guardrails, so the whole team's ceiling rises, not just mine.
